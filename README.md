# Sno
Sno is one of my older Scratch games all the way back in 2023. It's a remake of one of my friend's games, sharing the same name as well. The game involves you avoiding the titular Snoman [sic] for as long as possible, with some additional surprises. The only controls are WASD that do what WASD does best. Both a game link and download are provided, have fun if you dare.
